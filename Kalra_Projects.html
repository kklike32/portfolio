Scratch Project

2018 September

In 9th grade at DHS, my partner Matthew Kang and I had to create a Scratch project in Computer Science Principles. We decided to make a hybrid mini-game and story in scratch. Matthew and I decided to create a model the game of a student running away from school. The user can make decisions on whether they want to escape school or stay there. If the user decides to run away, it will turn into a game. From then onwards, the user has to use the arrow keys to dodge incoming school materials like pencils and pens. So far,Matthew and I have created two levels. You can see our project notebook here .
